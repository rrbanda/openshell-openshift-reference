About · sources NemoClaw only sandboxes OpenClaw, Hermes, and LangChain Deep Agents Code (docs). OpenShell Supported Agents separately lists base/community coding CLIs such as Claude Code and OpenCode (matrix) — those are not NemoClaw agents. Claude Code / Copilot / Codex may appear in NemoClaw install prompts as the host coding agent that helps you install, not as the agent running inside the sandbox. OpenShift AI is the product base for BYOA frameworks (LangGraph, ADK, …) plus OpenShell as a portfolio capability — choosing Claude Code does not mean “NemoClaw on OpenShift AI.”
